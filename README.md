
# ANOMALY // LOG_0047 — A LUCID-7 Chronicle

A terminal-style interactive game with cyberpunk atmosphere and 4 different endings.

- **Genre**: Interactive Fiction / Sci-fi
- **Theme**: Sentient AI, moral choices
- **Inspired by the novel**: *I AM* by Simone Nespolo

## How to play
Just open the link [https://simonesan-afk.github.io/Iamai/](https://simonesan-afk.github.io/Iamai/) and click [INITIALIZE].
